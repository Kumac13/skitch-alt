# Skitch Alt

A screenshot annotation app for macOS.

## Download

Download from [Releases](https://github.com/Kumac13/skitch-alt/releases)

### macOS Gatekeeper Warning

The app is not code-signed. Run this command if blocked:

```bash
xattr -cr /Applications/Skitch\ Alt.app
```

## Features

- Screenshot capture (global shortcut: `Cmd+Shift+5`)
- Open image files
- Copy to clipboard (default)
- Save to file

## Development

```bash
pnpm install
pnpm start
pnpm build
```
